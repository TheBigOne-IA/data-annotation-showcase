# 🧠 Custom GPTs for Data Annotation

This folder documents the custom GPTs I've created for AI-orchestrated annotation.

---

## 1. Sentiment Analysis Annotator

**Purpose:** Classify customer reviews as positive, negative, or neutral

**Input:** Text (customer reviews in bulk)

**Output:** CSV with columns: `text, sentiment`

**Example Use Case:** E-commerce product reviews, customer feedback forms

---

## 2. NER (Named Entity Recognition) Extractor

**Purpose:** Extract people, organizations, locations, and dates from unstructured text

**Input:** Text (legal documents, financial reports, contracts)

**Output:** JSON with entity categories

**Example Use Case:** Contract analysis, news article processing

---

## 3. Image Description & Tagging Assistant

**Purpose:** Generate descriptive tags and captions for images

**Input:** Image (uploaded via ChatGPT Plus)

**Output:** JSON with: `tags, caption, object_count`

**Example Use Case:** E-commerce catalog tagging, social media content organization
